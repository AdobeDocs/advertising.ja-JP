# 消費者のプライバシーに対応するAdobe Advertisingサポート

Adobe Advertisingでは、広告主が次の消費者プライバシー法に準拠できるよう、セキュリティとプライバシーの管理を提供しています。

* 一般データ保護規則（GDPR）

* カリフォルニア消費者プライバシー法（CCPA）

## 最新情報

以下の機能は新規または最近変更されました。

| 日付 | 機能 | 説明 | 詳細 |
| ---- | ------- | ----------- | -------------------- |
| 2025年3月10日（PT）リリース | プライバシー | （Advertising DSPのみ）「[!DNL Adobe's]」がDigital Advertising Alliance （DAA）との提携により、Advertising DSP エンドユーザーは、「Adobe Marketing Cloud - Advertising サービス」（DSP + Audience Manager Cookieを含むAdobe Experience Cloudのレガシー名）からハッシュ化されたメールアドレスに依存するすべての行動ターゲティングをオプトアウトできるようになりました。 以前、DSPではCookie ベースのオプトアウトのみがサポートされていました。<br><br> エンドユーザーが行動ターゲティングをオプトアウトすると、DSPはエンドユーザーのCookie、モバイル ID、またはハッシュ化された電子メールアドレス（[!DNL Unified ID 2.0 (UID2.0)] IDまたは[!DNL LiveRamp] [!DNL RampID]に関連付ける場合がある）をキャプチャします。 その後、DSPは、エンドユーザーがオプトアウト Cookieを5年間保持する限り、そのエンドユーザーに対する広告インプレッションに対する行動ターゲティングを除外します。<br><br> エンドユーザーは、[https://optout.aboutads.info](https://optout.aboutads.info)からブラウザー上の広告をオプトアウトするか、[https://youradchoices.com/control](https://youradchoices.com/control)からブラウザー、アプリ、またはトークン IDを使用してブラウザー上の広告をオプトアウトするかを選択できます。<br><br>広告主に作業は必要ありません。 | 「[Adobe プライバシーポリシー](https://www.adobe.com/privacy/policy.html)」を参照してください。 |
