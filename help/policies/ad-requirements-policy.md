# Adobe広告要件ポリシー

*最終更新日：2021 年 7 月 13 日*

*Adobe広告DSPおよびAdobe広告検索の場合*

この Ads Requirements Policy(「Policy」) は、Adobeの On-demand Services を通じて配信されるAdobe広告 (「Services」) の要件を規定します。 本ポリシーは、Adobeとお客様（「お客様」）との間で、オンデマンドサービスを使用する契約に基づいて組み込まれます。 本ポリシーで使用されるが、ここで定義されていない大文字の用語は、本契約に規定される意味を持ちます。

Adobeは、広告又は広告がリダイレクト又はリンク先のウェブサイトが本ポリシーの要件に違反するか否かを単独の判断で決定する権利を留保します。この場合、Adobeは、当該広告を拒否し、本サービスへのアクセスを停止又は停止します。

以下と契約に定める要件に加え、広告を配置する各メディアソースによって課される要件と、広告の配信に使用するサプライサイドプラットフォーム (SSP) または他のプラットフォームによって課される要件を満たす必要があります。 広告を配信するパブリッシャーおよびプラットフォームのポリシーを理解し、遵守するのは、お客様の責任です。

## 広告コンテンツの要件

### クリエイティブ要件

オンデマンドサービスを通じて送信、保存、表示、配布、またはその他の方法で提供されるすべての広告は、Adobeの広告仕様に含まれる広告仕様と一致している必要があります。これは、時々Adobeによって変更されます。

### 禁止されている広告

広告のリンクまたはリダイレクト先の Web サイトと広告に以下のいずれも含めたり、昇格させたりしないようにする必要があります。

* **無効なコンテンツ**. 広告は、違法なコンテンツを含んだり、違法な活動、製品、サービスを促進、促進、促進したり、促進したりしない可能性があります。 違法な薬物や麻薬品の使用（州レベルでの違法か連邦レベルかにかかわらず）や証券法違反を含むコンテンツ、製品、活動、サービスを含み、これらに限定されない。 本方針の目的上、大麻、麻又はこれらの物質の派生物 ( 大麻 (CBD) など ) に関するコンテンツ、製品、活動、又はサービスに関する広告は、以下の例外を除き、禁止されます。(1) 米国における麻由来の CBD 製品（他の CBD または THC 製品は除く）(2) 許可カナダ州の大麻製品で、(a) 既に適用Adobeライダーに参加しているもの、(b)Adobeのデューデリジェンス要件への準拠、及び (c) メディアサプライヤー及び適用法によって認められたもの。 「許可されたカナダの州」とは、アルバータ州、ブリティッシュコロンビア州、マニトバ州、ニューブルンズウィック州、ニューファンドランド州とラブラドール州、ノバスコシア州、オンタリオ州、プリンスエドワード島、サスカチュワン州を意味します。

* **他人の権利又は契約の侵害**. 広告は、他の広告の権利や契約に違反したり、侵害したり、これらの活動に関与する製品やサービスを促進したりすることはできません。 これには、知的財産権の侵害、プライバシー、宣伝、またはその他の個人または所有権の侵害（主に著作権侵害に使用されるファイル共有サービスの推進を含む）が含まれます。

* **暴力または危険な活動**. 広告は、衝撃的な内容、センセーショナルな内容、ゴリー、猥褻な内容を含む暴力的または危険な活動や資料を宣伝したり、表現したりすることはできません。

* **偽の広告または誤解を招く広告**. 広告に、誤ったコンテンツ、誤解を招くコンテンツ、または欺くコンテンツが含まれていない可能性があります。 これには、広告をクリックした後にユーザーが直接目的とする実際の宛先を隠す、別のサービスと誤ったサイトにリダイレクトまたはリンクする、システムの警告や警告（オペレーティングシステムやモバイルアプリケーションからなど）を模倣する、または別の製品やアプリケーションに似た広告が含まれます。 「無料」または割引された製品またはサービスのオファーには、法律に基づく要件に一致するオファーを要求するための条件、条件、またはその他の要件に関する明確かつ正確な開示を含める必要があります。 なされた主張は、正確で正直でなければなりません。

* **有害な広告。** 広告には、ウイルス、スパイウェア、マルウェア、またはそのようなソフトウェアや製品を含む Web サイトを含めたり、宣伝したりしない場合があります。 これには、不正なアクションを実行するように設計されたハッキングソフトウェア、不正なサービス、その他のサービスの促進が含まれます。 また、広告には、嫌がらせ、脅迫、軽蔑、中傷、差別的、不快、または他の好ましくないコンテンツを含めることはできません。

* **偽造品または偽造品**. 広告は、偽造文書（パスポートや公式の書類作業など）を含む偽造品や偽造品を宣伝または表現できません。

* **性的な示唆に富むコンテンツ**. 広告には、ヌディティ、ポルノ、アダルトサービスやエンターテイメント、暗示的または明示的な描写や言語など、性的な示唆に富むコンテンツを含んだり、宣伝したりしない場合があります。

* **オンライン薬局**. 広告は、処方薬や店頭薬をオンラインで購入できるようにする薬局サービスを宣伝することはできません。

* **医療**. 広告には、疾病や疾病の予防、治療、治療に有効であることを示す方法で（直接または暗黙の方法で）市場に出回る、政府の承認を受けていない製品やサービスに関する主張を含めることはできません。 広告は、政府または臨床試験の採用者によって許可またはライセンスを受けていない限り、消費者に対する直接的な汎用テストを宣伝することはできません。

* **タバコ**. 広告は、電子タバコを含むたばこ製品又は類型品の販売又は使用を提供し、又は促進することができない。

* **冒涜的で俗悪な言葉や猥褻な言葉。** 広告に不敬、俗悪、猥褻な言葉が含まれていないことがあります。 広告には、同じ効果を達成しようとする言語は含まれないが、不敬な言葉を隠す言語も含まれる。

* **憎むべき広告**. 広告は、人種または民族の起源、宗教、障害、医療または遺伝的状態、年齢、国籍、退役軍人の地位、難民の地位、入国者の地位、性的指向、性別、性的同一性その他の制度的差別または社会的差別に関連する特性に基づいて、個人または集団を非難してはならない。

* **機密イベント**. 広告は、自然災害、紛争、死、疫病、パンデミック、悲劇的な出来事、または公共の恐怖に対する合理的な感受性を生かし、または欠如している可能性があります。

* **ピラミッドスキーム**. 広告は、ピラミッドスキーム、マルチレベルマーケティングの機会、チェーンレター、その他の「リッチクイックを得る」スキームを宣伝することはできません。 収益を生み出す機会を含む広告には、ビジネスモデルの完全な説明を含める必要があります。

* **身体部品の販売**. 広告は、身体の部分や体液の販売を促進しない場合があります。

* **広告の気を散らす**. 自動再生機能 ( 自動再生機能 (Ads)、点滅、揺れ、鳴らすコンテンツ、パルス、点滅、過度にアニメーション化されたオブジェクト、テキスト、画像、ローカル共有オブジェクト (Flash Cookie やHTML5 ローカルストレージを含む )、ユーザーエクスペリエンスを中断する広告など。

* **政治コンテンツ**. 政治コンテンツには、次のようなコミュニケーションが含まれます。(1) 任意の候補者（連邦、州、地方、国際）又は候補者の選挙運動に対して支払われる。(2) 任意の管轄権（連邦、州、地方または国際）のキャンペーンファイナンス法に基づいて登録又は報告を受けた者が支払う。(3) 明確に識別された任意の候補者（連邦、州、地方、国際）を明示的に支持するか、それに反対するか、または (4) その他の場合には、任意の管轄権のキャンペーンファイナンス法の下に規定されます。 この禁止には、選挙、政治運動、公職候補、投票、投票イニシアティブ、法制、国民投票に直接的または間接的に関連する発行のアドボカシー広告が含まれますが、これらに限定されません。

### 制限付き広告

以下のカテゴリのコンテンツを含む広告は、以下に説明する特定の制限を受けて許可されます。 制限は、法規制、業界のベストプラクティス、その他のビジネス上の考慮事項に基づいています。

* **アルコール**. アルコール飲料やアルコール会社を宣伝または宣伝する広告は、適用法や自主規制の規則に従う場合にのみ許可されます。 例えば、アルコール広告は未成年者を対象とせず、アルコールの広告を許可する国でのみ実行できます。 しかし、どのような広告も、過剰なアルコール消費を誇示したり、表現したりすることはできません。

* **賭博**. カジノ、宝くじ、その他のチャンスのゲーム（オンラインかオフラインかにかかわらず）の促進を含む賭博関連のコンテンツを含む広告は、適用法に従ってのみ促進することができます。 しかし、広告は過剰な賭博行動を誇示したり、表現したり、賭博のオッズやリスクを誤表したりすることはありません。

* **製薬広告**. 処方薬と店頭薬の宣伝を行う広告は、適用される連邦法および地方法、規制、ガイドラインに準拠する必要があります。

* **金融商品・サービス**. 金融商品やサービスを消費者に宣伝する広告は、適用される連邦法や地域の法律、規制、ガイドラインに準拠し、適用されるすべての料金を開示する必要があります。 広告は、支払日ローン、ローンの発行日から 60 日以内に全額返済を必要とするローン、タイトルローン、質問屋からの支払日ローン、または質問屋を促進することができません。 住宅ローン、自動車ローン、学生ローン、信用の極度化に関する広告は、返済期間の最小・最大期間、年率、ローン総額の代表例、手数料を含む主要な条件を顕著に開示する宛先 URL に対して許可されます。

* **暗号通貨とブロックチェーンサービス**. 暗号通貨やその他のブロックチェーン関連の製品やサービスを宣伝する広告は、適用される連邦法や地域の法令、規制、ガイドラインに準拠し、未登録の証券の提供、非代替トークン、特定の暗号トークンの販売や発行に関連する製品やサービスを促進してはなりません。 暗号通貨交換プラットフォームおよびウォレットサービスの広告は、すべての適用可能な料金が明確に開示され、広告されたサービスプロバイダが連邦登録のマネーサービスビジネスである場合に許可されます。 トランザクション監視やその他のコンプライアンス機能をサポートするブロックチェーン分析サービスプロバイダー向けの広告が許可されています。

* **減量に関する広告**. ヒト絨毛性ゴナドトロピン (hCG) を含む製品を含む、減量または食欲抑制薬およびサプリメントを促進する広告は認められていません。 広告は、万一、危険な減量の要求を促進しないか、または、前後の減量のイメージを欺く可能性があります。

### 未成年者に対する広告

広告は、16 歳未満のユーザーを意図的にターゲットにするために個人データを使用しない場合があります。 上記の禁止広告および制限広告に加え、広告は、18 歳未満のユーザーに対し、知識的に、次の製品またはサービスを販売または宣伝することはできません。

* アルコール飲料及び関連アクセサリー
* 発射体、BB、またはペレット銃/デバイス
* ハンドガンの安全証明書
* 特性を落とすことが可能なエアゾール塗料又はエッチングクリーム
* サルビア・ジヴィノラムまたはサルビノリン A を含む物質/材料
* 栄養補助食品
* 紫外線日焼け装置における日焼け
* 賭博商品とサービス（宝くじを含む）
* 刺青、ボディピアシング、または永久的な化粧品などのボディブランディング

## ターゲット広告

本サービスは、ユーザーに対する広告のターゲット設定や、Web サイトや他のオンラインサービスへの訪問に基づくユーザーの再ターゲット化を含む、ユーザーに対する広告のターゲット設定を行わない場合があります。

* **禁止されたターゲティング**. 広告は、既知の要素と推測される要素のどちらに基づいても、オーディエンスに対してターゲット設定またはターゲット設定されない場合があります。

   * 宗教的または類似の信念または関連
   * 人種、色、民族の起源
   * 性史、興味または指向
   * トランスジェンダー ID
   * 遺伝情報または生体情報
   * 負の財務ステータス（クレジットスコアなど）、または犯罪記録、履歴、有罪判決
   * 健康又は医療記録（処方箋を含む。）
   * 個人の苦難に関する関係または関係のステータス（離婚、忌避など）
   * 犯罪、虐待、または心的外傷の被害者としての地位を含む虐待と外傷
   * 社会的地位、移民、難民の地位に基づくものを含め、社会的地位または脆弱なグループのメンバーシップ

* **欧州連合**. また、EU では、ユーザーに関する既知または推測される次の要因に基づいて、広告がオーディエンスにターゲット設定または向けられない場合があります。

   * 政治的所属
   * 労働組合メンバーシップ
   * 個人データのその他の特別なカテゴリ

* **ヘルス関連のターゲティング**. 本サービスを利用して、機密性の高い健康関連データを収集したり、ユーザーの機密性の高い健康や医療に関する推測を行ったりすることはできません。 特に、以下のいずれに対しても、本サービスを使用して広告をターゲットにすることはできません。がん、精神的な健康関連の状態、性的に感染した疾患の総称。 広告は、にきび、アレルギー、歯科、視覚、胸焼け、風邪とインフルエンザ、洞、頭痛、背中の痛み、応急処置、喉の痛み、痛み、血糖管理、食事とフィットネス、脱毛、ビタミンとサプリメントを含む、非感受性の健康状態を対象とする場合があります。

## 差別なし

本サービスを利用して、特定の人々のグループを広告用に誤ってターゲティングし、特定の人々のグループを誤って広告を見ない、または広告に差別的な内容を含む、法律に違反して差別を行うことはできません。 お客様は、保護されたクラスに対する差別を防ぐために、サービスを責任を持って使用する責任を負います。たとえば、広告が保護されたクラスに異なる影響を及ぼさないようにするなどです。
