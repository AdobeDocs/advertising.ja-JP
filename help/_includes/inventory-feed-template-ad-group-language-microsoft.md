# テキスト広告テンプレート — 広告グループレベルの言語

**[!UICONTROL Language]:** 広告グループ内の広告のターゲット言語： *英語*, *フランス語*, *フィンランド語*, *ドイツ語*, *ノルウェー語*, *スペイン語*&#x200B;または *スウェーデン語*. 新しいキャンペーンのデフォルトはです。 *英語*.

この設定は、広告を表示できる国と地域を決定します。 キャンペーンのロケーションターゲットと互換性のある言語を選択します。
