# [!DNL LY Ads] キャンペーンおよび広告グループ設定の「否定的なキーワード」フィールドに関する注意

>[!NOTE]
>
>* 最大長はキーワードあたり80文字です。
>* キャンペーンと広告グループの設定または[!UICONTROL Keywords] > [!UICONTROL Negatives] ビューから、キャンペーンと広告グループのレベルで負のキーワードを設定できます。 キャンペーンレベルのネガティブは、広告グループレベルで適用されます。
