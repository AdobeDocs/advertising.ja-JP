# Yahoo！の「除外キーワード」フィールドに関する注意 キャンペーンと広告グループの設定

>[!NOTE]
>
>* 最大長は、1 つのキーワードにつき 80 文字です。
>* 除外キーワードは、キャンペーンおよび広告グループの設定から、または [!UICONTROL Keywords] > [!UICONTROL Negatives] 表示 キャンペーンレベルの除外は、広告グループレベルで適用されます。
