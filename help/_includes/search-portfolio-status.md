# Portfolio ステータス

<!-- used in multiple procedures -->

* *ドラフト：* ポートフォリオに、目的がないか、少なくとも1つの割り当てられたアクティブなキャンペーンがありません。 このステータスは自動的に設定され、欠落している設定を割り当てて変更を保存すると、自動的に&#x200B;*非アクティブ*&#x200B;に変わります。

* *非アクティブ：*&#x200B;検索、ソーシャル、およびCommerceは、レポート用に関連するキャンペーンのコスト/クリック/インプレッションデータを収集します。

* *アクティブ：* （すべての必要な設定を持つ既存のポートフォリオ。少なくとも1つのアクティブな入札単位を持つアクティブなキャンペーンが必要） Search, Social, &amp; Commerceは、関連するキャンペーンのコスト/クリック/インプレッション データと収益データを収集し、データもモデル化します。 非アクティブなポートフォリオをアクティブにしてデータのモデリングを開始するか、最適化されたポートフォリオをアクティブ状態にダウングレードして、最適化を停止します。

* *最適化：* （すべての必要な設定を持つ既存のポートフォリオ。少なくとも1つのアクティブな入札単位を持つキャンペーンが必要） Search, Social, &amp; Commerceは、関連するキャンペーンのコスト/クリック/インプレッションデータと収益データを収集し、データをモデル化して最適化戦略を決定し、ポートフォリオのタイプとキャンペーン入札戦略に必要に応じてポートフォリオを最適化します。 ステータスを「最適化」に変更することは、「ポートフォリオの起動」とも呼ばれます。 キーワードと広告の入札、（有効な場合）入札調整値、（有効な場合）キャンペーン予算、入札戦略目標を最適化できます。 ポートフォリオに、少なくとも1つのアクティブな広告とキーワード/プレースメントを含む少なくとも1つのアクティブなキャンペーンが含まれていない限り、ステータスを「最適化」に変更することはできません。 **メモ：**&#x200B;最適化されたポートフォリオからすべてのキャンペーンを削除するか、それらを削除すると、ポートフォリオの状態が自動的に「非アクティブ」に変更されます。
