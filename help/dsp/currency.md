# Adobe Advertisingでサポートされている通貨

Adobe Advertisingは以下の通貨をサポートしています。

>[!NOTE]
>
>Adobe Analyticsでは、[より大きな通貨セット &#x200B;](https://experienceleague.adobe.com/docs/analytics/implementation/vars/config-vars/currencycode.html)をサポートしています。

| コード | 通貨名 |
| ------ | -------------- |
| AUD | オーストラリアドル |
| BRL | ブラジル・レアル |
| CAD | カナダ ドル |
| CLP | チリ・ペソ |
| CNY | 中国人民元 |
| DKK | デンマーク クローネ |
| EUR | ユーロ |
| GBP | イギリス ポンド |
| HKD | 香港ドル |
| INR | インドルピー |
| 円 | 日本円 |
| MXN | メキシコペソ |
| MYR | マレーシア・リンギット |
| NZD | ニュージーランドドル |
| RUB | ロシア ルーブル |
| SGD | シンガポール・ドル |
| THB | タイバーツ |

>
> [!DNL DSP]では、CAD、DKK、EUR、GBP、HKD、JPY、NZD、SGD、USDの通貨でのみ請求が可能です。

## [!DNL Adobe Advertising Search, Social, & Commerce]

サポートされている検索エンジンでサポートされているすべての通貨。 これは、[&#x200B; [!DNL Google Ads]によってサポートされている](https://developers.google.com/adwords/api/docs/appendix/codes-formats#currency-codes)個と同じです。
