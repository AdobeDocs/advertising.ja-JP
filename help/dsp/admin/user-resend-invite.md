# 保留中のユーザー招待を再送信する

*管理者ユーザーのみ*

管理者ユーザーは、組織アカウントの保留中の招待を持つユーザーに招待を再送信できます。 招待リンクは24時間以内に期限切れになります。

1. 次のいずれかの操作を行います。

   * メインメニューで、**[!UICONTROL Settings]** > **[!UICONTROL Users]**&#x200B;をクリックします。

   * メインメニューで、**[!UICONTROL Settings]** > **[!UICONTROL Account]**&#x200B;をクリックし、**[!UICONTROL Edit users]**&#x200B;をクリックします。

1. **招待状** セクションで、![再送信](/help/dsp/assets/resend.png)をクリックします。
