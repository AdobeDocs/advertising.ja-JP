# 保留中のユーザーの招待を解除

管理者ユーザーは、保留中の招待を持つユーザーの招待を解除できます。 ユーザーを非招待すると、指定された電子メールアドレスに送信された招待リンク内のトークンが無効になります。

1. 次のいずれかの操作を行います。

   * メインメニューで、**[!UICONTROL Settings]** > **[!UICONTROL Users]**&#x200B;をクリックします。

   * メインメニューで、**[!UICONTROL Settings]** > **[!UICONTROL Account]**&#x200B;をクリックし、**[!UICONTROL Edit users]**&#x200B;をクリックします。

1. **[!UICONTROL Invitations]** セクションで、招待の横にある&#x200B;**[!UICONTROL X]**&#x200B;をクリックします。
