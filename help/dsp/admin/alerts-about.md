# アラート配信について

サブスクリプションと呼ばれる自動メールアラートのグループを作成し、指定されたアカウント、キャンペーン、広告、プライベート在庫取引のパラメーターが満たされたときにメール通知を送信することができます。

例えば、キャンペーンが指定された金額の予算を費やした場合にアラートを受け取ることができます。 また、過去N日間にプライベート在庫取引が指定された成約率を下回った場合にもアラートを受け取ることができます。

サブスクリプションにアラートを追加または削除したり、サブスクリプションのメール通知を受け取るユーザーを編集したり、アラートのサブスクリプションを無効または再度有効にしたり、アラートのサブスクリプションを削除したりできます。

>[!TIP]
>
> キャンペーンレポートを有効に確認していない場合は、キャンペーンのローンチ後できるだけ早くメールレポートを設定します。

## 利用可能なアラート

* アカウント

   * 口座残高

   * アカウント支出

   * 新規キャンペーンが作成されました

* キャンペーン

   * キャンペーン費用なし

   * キャンペーン費用

   * キャンペーン支出%上限

   * キャンペーンインプレッションなし

   * キャンペーン eCPM

* 広告

   * 広告の却下

* フィード（Private Inventory Deals）

   * 成約率

   * フィード入札率

   * フィードへの支出がありません

   * フィードインプレッションがありません

   * デフォルトの配置のない保証フィード
