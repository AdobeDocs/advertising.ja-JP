# 接続されたテレビのリーチ計画の予測のエクスポート

[!UICONTROL Budget vs Reach] 列と [!DNL Microsoft Excel] 列を使用して**[!UICONTROL Budget] 予測を [!UICONTROL Reach] スプレッドシートに書き出すことができます。

* 生成された予測で、右上の **[!UICONTROL ...]** > **[!UICONTROL Export]** をクリックします。
