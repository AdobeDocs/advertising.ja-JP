# 接続されたテレビのリーチ計画の予測のエクスポート

[!UICONTROL Budget] 列と [!UICONTROL Reach] 列を使用して**[!UICONTROL Budget vs Reach] 予測を [!DNL Microsoft Excel] スプレッドシートに書き出すことができます。

* 生成された予測で、右上の **[!UICONTROL ...]** > **[!UICONTROL Export]** をクリックします。
