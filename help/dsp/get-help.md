# ヘルプを表示

オンラインヘルプは、ユーザーインターフェイス内の任意のページから開くことができます。 オンラインヘルプが質問に答えない場合は、お問い合わせください。

## オンラインヘルプを開く

すべての機能の使用方法を説明する完全なヘルプセットを開くには、メインメニューの「**[!UICONTROL Help]**」をクリックします。

## 連絡先 [!DNL Adobe]

DSPの製品やアカウントの問題については、Adobe アカウントチームにお問い合わせください。

問題を再現する詳細な手順、関連データ、実行した予備のトラブルシューティングに関する情報を提供します。 ブラウザー名とバージョンも含めます。

レポートの問題の場合は、レポートを削除せずに、レポート名を指定します。
