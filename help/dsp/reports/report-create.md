# カスタムレポートの作成

1. メインメニューで、**[!UICONTROL Reports]**/**[!UICONTROL Custom Reports]** をクリックします。

1. 右上で、「**[!UICONTROL Create]**」をクリックします。

1. [ レポート設定 ](/help/dsp/reports/report-settings.md) を指定します。

1. 「**[!UICONTROL Create Custom Report]**」をクリックします。
