# カスタムレポートの編集

1. メインメニューで、「**[!UICONTROL Reports]**」をクリックします。
1. 次のいずれかの操作をおこないます。
   * レポート名をクリックします。
   * レポート行の [!UICONTROL Options] 列で、**[!UICONTROL options]**/**[!UICONTROL Edit]** をクリックします。
1. [ レポート設定 ](/help/dsp/reports/report-settings.md) を編集します。
1. 「**[!UICONTROL Edit Custom Report]**」をクリックします。
