# ヘルプを表示

<!-- Can remove this page when we move this into DSP help -->

AI支援のクエリをすべての[!DNL Creative]および（Advertising DSPを使用する広告主） [!DNL DSP]のドキュメントに送信するか、[!DNL Creative]の完全なユーザーガイドを開くことができます。 ご質問にお答えできない場合は、お問い合わせください。

## AIを活用したチャットインターフェイスで情報を検索

「[AI支援チャットインターフェイスを使用して製品ドキュメントを検索する](/help/dsp/agent-chat.md)」を参照してください。

## ユーザーガイドを読む

すべての機能の使用方法を説明する完全なユーザーガイドを開くには、メインメニューの「**[!UICONTROL Help]**」をクリックし、「[!DNL Creative]」章を開きます。

## [!DNL Adobe]へのお問い合わせ

製品またはアカウントの問題については、次の操作を行います。

* （セルフサービス契約を持つ広告主） [https://experienceleague.adobe.com/home#support](https://experienceleague.adobe.com/home?support-tab=home#support)で組織のチケットを開きます。

  製品に「[!UICONTROL Advertising - Creative]」を選択します。

* （フルサービス契約を有する広告主）Adobeアカウントチームにお問い合わせください。

チケットまたはコミュニケーションで、問題を再現するための詳細な手順、関連データ、ブラウザー名とバージョン、実行した予備トラブルシューティングに関する情報を提供します。 レポートの問題については、レポートを削除せず、レポート名とURLを指定してください。
