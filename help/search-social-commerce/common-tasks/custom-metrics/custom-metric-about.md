# カスタム指標について

カスタム（派生）指標を設定できます。この指標は、標準トラフィック指標と、広告主が追跡する既存のコンバージョン指標から計算されます。 カスタム指標は、`[Cost]/[Registrations]`などの単純なものから、必要に応じて複雑なものまでです。

カスタム指標は、ポートフォリオビューやキャンペーン管理ビュー、レポートの列として含めることができます。

>[!NOTE]
>
>Adobe Analyticsの計算指標と高度な計算指標は、計算で使用できません。
