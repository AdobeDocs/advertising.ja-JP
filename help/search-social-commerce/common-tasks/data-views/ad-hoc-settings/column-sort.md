# 列の並べ替え

<!-- The same in new UI and legacy CM views -->

1. （使用可能な場合）列見出しの右側で、![下向き矢印](/help/search-social-commerce/assets/arrow-down-expand.png "下向き矢印")をクリックします。

1. 並べ替えオプションを選択します。

   * *昇順で並べ替え* （A ～ Z、または1 ～ 10）

   * *降順*&#x200B;で並べ替え（ZからAまたは10から1）
