# 列の並べ替え

<!-- The same in new UI and legacy CM views -->

1. （使用可能な場合）列見出しの右側にある ![ 下矢印 ](/help/search-social-commerce/assets/arrow-down-expand.png " 下矢印 ") をクリックします。

1. 並べ替えオプションを選択します。

   * *昇順に並べ替え* （A ～ Z、または 1 ～ 10）

   * *降順に並べ替え* （Z から A、または 10 から 1）
