# ロールについて

役割は、ユーザータイプが実行できる権限を決定します。 各ユーザーは役割に割り当てられ、役割が許可するすべてのアクティビティを実行できます。

次の条件を持つユーザーのみ [!DNL Adobe] アカウントマネージャーと管理者の役割は、ユーザーに役割を割り当てることができます。
