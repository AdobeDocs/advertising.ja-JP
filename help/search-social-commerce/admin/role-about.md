# 役割について

ロールは、ユーザータイプが実行できる権限を決定します。 各ユーザーは役割に割り当てられ、その役割が許可するすべてのアクティビティを実行できます。

アカウントマネージャーと管理者 [!DNL Adobe] 役割を持つユーザーのみがユーザーに役割を割り当てることができます。
