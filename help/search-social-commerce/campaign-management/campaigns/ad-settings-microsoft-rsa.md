# [!DNL Microsoft® Advertising] レスポンシブ検索広告設定

レスポンシブ検索広告形式は、検索ネットワーク上の広告で使用できます。 広告ネットワークは、広告要素の最も効果的な組み合わせを使用して、レスポンシブ検索広告を動的に構築します。

>[!NOTE]
>
>各広告グループには、最大 3 つの有効なレスポンシブ検索広告を含めることができます。
