# [!DNL Microsoft Advertising]件のレスポンシブ検索広告の設定

検索ネットワーク上の広告では、レスポンシブ検索広告の形式を利用できます。 広告ネットワークは、広告要素の最も効果的な組み合わせを使用して、レスポンシブ検索広告を動的に組み立てます。

>[!NOTE]
>
>各広告グループには、有効なレスポンシブ検索広告を3つまで含めることができます。
