# [!DNL Baidu] テキスト広告の設定

>[!NOTE]
>
>[!DNL Baidu]広告の広告コピーを変更すると、既存の広告が削除され、同じプロパティを持つ新しい広告が作成されます。

## [!UICONTROL Text Ad]

**[!UICONTROL Ad Title]:**&#x200B;広告の見出し。 最大長は、14個のダブルバイトまたは28個のシングルバイト文字です。

**[!UICONTROL Description Line 1]:**&#x200B;広告の本文の最初の行。 最小長は4つの2 バイトまたは8 シングルバイト文字で、最大長は20 ダブルバイトまたは40 シングルバイト文字です。

**[!UICONTROL Description Line 2]:**&#x200B;広告の本文の2行目。 最小長は4つの2 バイトまたは8 シングルバイト文字で、最大長は20 ダブルバイトまたは40 シングルバイト文字です。

**[!UICONTROL Display URL]:**&#x200B;広告に表示されるURL。 最大長は35文字です。

**[!UICONTROL Mobile Preferred]:** （オプション） ネットワークが、デスクトップ ユーザーやタブレット ユーザーではなく、モバイル デバイス ユーザーに広告または広告拡張機能を表示しようとすることを許可します。 デフォルトでは、このオプションは有効になっておらず、広告はあらゆるデバイスタイプに表示されます。

>[!NOTE]
>
>ネットワークは、広告または広告拡張機能が好みのデバイスタイプで表示されることを保証するものではありません。
