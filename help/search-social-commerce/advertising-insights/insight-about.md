# について [!UICONTROL Advertising Insights]

[!UICONTROL Advertising Insights] アクティブなキャンペーンを含む最適化されたポートフォリオとアクティブなポートフォリオに関する、視覚的で実用的なデータを提示します。

各インサイトはオンデマンドで生成され、出力は、ダウンロード可能なファイルか画面上のレポートのどちらかです。 インサイトファイルの生成済みインスタンスを削除できます。

## のタイプ [!UICONTROL Advertising Insights]

| Insight | 説明 |
| --- | --- |
| [!UICONTROL AMO-AA Tracking Discrepancy] | (Adobe AdvertisingとAdobe Analyticsの統合のみの広告主 ) トラッキングの問題を識別します。この問題は、日別の特徴で、通常の平方偏差以外のクリックトゥインスタンスの比率または 20%以上の平方偏差を持ちます。 レポートには以下が含まれます。<ul><li>A [!DNL Microsoft® PowerPoint] ファイルに、データの相違の概要、最もトラッキングの問題が多いキャンペーンのキャンペーン別の相違、トラブルシューティングの提案、レポート手法の説明を示す表およびトレンドグラフを表示します。</li><li>A [!DNL Microsoft® Excel] ファイルには、ポートフォリオ内の各キャンペーンの概要データと、各キャンペーンの過去 1 か月の日別データが含まれます。</li></ul> |
| [!UICONTROL Attribution Analysis] | ([!DNL PowerPoint] プレゼンテーション形式を参照 ) 異なるアトリビューションモデルが、単一のポートフォリオに対して収益モデルの改善と最適化をいつ実現できるかを示します。 |
| [!UICONTROL Campaign Caps] | ([!DNL PowerPoint] プレゼンテーション形式 ) 過去 30 日間の単一ポートフォリオの支出がキャンペーン予算の上限によって制限されているかどうかを示し、投資利益率を最適化するためにポートフォリオ設定の調整を推奨します。 |
| [!UICONTROL Day of Week] | ([!DNL PowerPoint] プレゼンテーション形式 ) 過去 30 日間の、1 つのポートフォリオの曜日別 (DOW) のパフォーマンスを示し、DOW 支出目標を推奨して投資回収率を高めます。<br><br>**注意：** Portfolioが十分な費用がない場合や、過去 2 日間にターゲティングに費やすことができなかった場合は、インサイトに使用できません。 |
| [!UICONTROL Delayed Revenue] | ポートフォリオのコンバージョンの遅延（広告クリックから後続のコンバージョンまでの経過時間）を測定し、 [重み付け売上高](/help/search-social-commerce/glossary.md#w-x)、ROI、およびラグによるモデルの精度。 この分析では、クリック日属性のパフォーマンス指標の最近の変更を検出し、クリック日とトランザクション日によるレポートの影響を確認できます。<br><br>このインサイトには、 [!DNL Excel] ワークブックに重み付けされた売上高、重み付けされていない売上高、売上高の精度およびラグ要因に関する個々のシートが含まれます。 また、 [!DNL Powerpoint] 様々なグラフを含むファイル。 |
| [!UICONTROL Event Path] | イベントパスとマルチタッチコンバージョンを分析することで、様々なチャネルやキャンペーンがコンバージョンをどのように促進しているかを識別します。 ファイルを XLSX および ZIP（圧縮 XLSX）形式でアップロードし、a) イベントを抽出する、b) 分類されたイベントを分析することができます。 |
| [!UICONTROL Google Account Audit] | ( の [!DNL Google Ads] アカウント（プリセールスに最も役立つ）アカウントのパフォーマンスの概要と、アカウントがどの程度効果的に管理されているかを示す欠陥を強調します。 監査には、入札、一致タイプ、予算上限、曜日パフォーマンス、モバイルパフォーマンスなどに関する情報が含まれます。 インサイトを生成するには、1) をアップロードする必要があります。 [!DNL Google Ads] アカウントのキャンペーン、キーワード、変更履歴レポートを [!DNL Google Ads] web ユーザーインターフェイスおよび b) [!DNL Google Ads Editor] アプリケーション。 すべてのファイルは、CSV、TSV、TXT、ZIP（圧縮 CSV、TSV、TXT）形式である必要があります。<br><br>このインサイトは、見込み客が検索、ソーシャル、コマースがパフォーマンスを高め、分析で強調表示された不備に対処する方法を理解するのに役立ちます。 また、Adobeアカウントチームは、顧客を正式にオンボーディングする前に、Insight を使用してアカウントを評価することもできます。 |
| [!UICONTROL Impression Share Lost] | ([!DNL PowerPoint] プレゼンテーション形式 ) ポートフォリオの予算でのインプレッション共有が制限されている場合に示します [!DNL Google Ads] キャンペーンを作成し、予算とキャンペーンの変更を推奨します。[!UICONTROL Multiple]&quot;設定に応じて、 |
| [!UICONTROL Location Target Performance] | ( の [!DNL Google Ads] キャンペーン； [!DNL Excel] スプレッドシート形式 ) 入札調整、クリック数、コスト、 [重み付け売上高](/help/search-social-commerce/glossary.md#w-x) を 1 つのポートフォリオ内の各ロケーションターゲットに対して設定します。 過去 90 日以内の任意の日付範囲でデータを利用でき、各キャンペーンの日別の結果や概要を表示できます。 |
| [!UICONTROL Match Type] | コストと [重み付け売上高](/help/search-social-commerce/glossary.md#w-x) 単一のポートフォリオ内の一致タイプを、過去 90 日間まで保持できます。 |
| [!UICONTROL Mobile Optimization] | ([!DNL PowerPoint] プレゼンテーション形式 ) 過去 30 日間の、デバイスタイプ別の単一ポートフォリオのパフォーマンスを表示し、モバイル固有の最適化機能の現在の設定を評価し、設定の調整を推奨します。 |
| [!UICONTROL Model Accuracy] | 検索、ソーシャル、コマースが、単一のアクティブなポートフォリオまたは最適化されたポートフォリオの支出と売上高をどの程度予測し、モデルの不正確なソースを特定しているかを識別します。 レポートには以下が含まれます。<ul><li>A [!DNL PowerPoint] 前の 7 日間の予測および実際のコスト、クリック数、クリック単価 (CPC)、売上高、クリックあたりの売上高、ROI (ROI) の概要を含むファイル。前月の超過支出および過少支出単位に関する集計データ、コストモデルの正確性、売上高モデルの正確性、予測コスト、CPC、売上高、RPC、ROI を示す実際の値、クリックレベルによるモデル精度、最も支出が多い（超過している）入札単位、およびモデルの精度を向上させるための推奨事項。</li><li>An [!DNL Excel] ファイルに含める必要があります。</li></ul>このインサイトでは、ポートフォリオレベルのモデル精度レポートよりも詳細な情報が提供されます。このレポートは、 [!UICONTROL Portfolios] 表示。 |
| [!UICONTROL Normalized Sim (Combined)] | 複数のポートフォリオに対する、指定した数の目標支出レベル（手順）に対する最適な支出配分を返します。 このインサイトは、正規化された [重み付け売上高](/help/search-social-commerce/glossary.md#w-x) 指定した期間の週別シミュレーションに基づく。 a) 選択した各ポートフォリオに対して個別のシミュレーションを生成するか、b) 同じ目的と通貨を持つ複数のポートフォリオにまたがる単一のシミュレーションを生成できます。 シミュレーションはスプレッドシートで表示されます。<br><br>単一の複数ポートフォリオのシミュレーションの場合、スプレッドシートには次の 2 つのシートが含まれます。1) 各目標支出レベル (step) の正規化された重み付け売上高を示すトレンドグラフ、2) 該当する支出レベル (step) ごとに 1 行を示す表。 個々のポートフォリオのシミュレーションに関して、スプレッドシートには、該当するすべてのポートフォリオと期間をリストした概要シートと、各ポートフォリオの期間を 2 枚表示した概要シートが含まれます。a) 各目標支出レベル (step) と b) の正規化された加重売上高を示す傾向グラフ。 |
| [!UICONTROL Portfolio Launch] | ([!DNL PowerPoint] プレゼンテーション形式 ) ポートフォリオを起動する準備ができているかどうかを識別します（「最適化済み」ステータスに変更）。 レポートには、予測されるパフォーマンスと実際のパフォーマンスの比較、推奨ポートフォリオの開始日と、コストと売上高の予測から除外する必要のある日付のリスト、コスト、売上高、コストあたりの売上高、時間の経過に伴うキーワードの変換、過去 2 週間の支出を制限したキャンペーンに関する情報、開始または未開始のレコメンデーションが含まれます。 |
| [!UICONTROL Quality Score] | インプレッション数のあるキーワードの品質スコアのトレンドを次のいずれかで表示 [!DNL Google Ads] または [!DNL Microsoft® Advertising] 過去 90 日間 a) a [!DNL PowerPoint] ファイルと b) [!DNL Excel] 最も費用の高い 100 個のキーワードと、（該当する場合）品質の低い上位 100 個のキーワードを含むファイル。<br><br>**メモ：**<ul><li>ポートフォリオに 20,000 を超える入札単位がある場合、（90 日ではなく）過去 30 日間に複数回クリックしたキーワードのみが含まれます。</li><li>多くのキーワードのデータを含むインサイトの生成に時間がかかります。</li></ul> |
| [!UICONTROL Query Cross Matching] | ([!DNL Google Ads] アカウント ) [!DNL Google Ads] 複数のキーワードに一致し、トラフィックを誘導する場所の候補を提供します。 サーチクエリを使用すると、キーワードのパフォーマンスが向上し、その結果、モデルの精度が向上します。<br><br>このインサイトには、クロスマッチクエリ、一致キーワード、パフォーマンスデータのスプレッドシート、ネットワーク図が含まれ、クロスマッチクエリとキーワードの関係、基になる検索クエリレポート、データの解釈方法に関する情報を視覚化できます。 |
| [!UICONTROL Settings Audit] | ポートフォリオがベストプラクティスに従って設定されているかどうかを示します。 以下が含まれます。<ul><li>A [!DNL PowerPoint] ファイルに含まれるのは、すべてのポートフォリオとそのコンポーネントキャンペーンの主要設定が、ベストプラクティスに従って設定されているか、キャンペーンがポートフォリオに割り当てられていないか、およびベストプラクティスから逸脱した結果です。</li><li>An [!DNL Excel] ファイルには、主要なポートフォリオ設定と主要なキャンペーン設定ごとに別のタブが表示されます。 データには、ポートフォリオごとのポートフォリオ設定と、ベストプラクティスを満たさない各キャンペーンのキャンペーン設定が含まれます。</li></ul> |
| [!UICONTROL Time-of-day Analysis] | ( [!DNL Google Ads] 検索、表示、または買い物キャンペーンのみ ) 提案 [!DNL Google Ads] 1 つのポートフォリオの 1 時間ごとのパフォーマンスに基づく、1 日の異なる時間（広告スケジュール入札修飾子）のキャンペーンレベルの入札修飾子。<br><br>このインサイトには、 [!DNL Excel] 個々のシートを含むワークブック [!UICONTROL Detailed Performance] （キャンペーンごとの期間別パフォーマンス） [!UICONTROL Recommended TBAs] （各キャンペーンの推奨広告スケジュール）。 次の項目を書き出すことができます。 [!UICONTROL Recommended TBAs] 直接に [!DNL Google Ads] 編集者です。 このインサイトには、 [!UICONTROL Powerpoint] ファイルに、補助的なデータと説明が含まれています。 |
