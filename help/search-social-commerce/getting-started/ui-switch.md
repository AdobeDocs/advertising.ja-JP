# 新しいユーザーインターフェイスと従来のユーザーインターフェイスの切り替え

新しい最適化ワークフロー<!-- default optimization workflow -->を含む新しいユーザーインターフェイス （UI）と、従来の最適化ワークフローとすべての追加機能を含む従来のUIを切り替えることができます。 ログインすると、新しいUIがデフォルトで開きます。

## 新しいUIから従来のUIへの切り替え

* ビューの右上で、![古いUIに切り替え](/help/search-social-commerce/assets/switch-to-old-ui.png "古いUIに切り替え") **[!UICONTROL Switch to Old UI]**&#x200B;をクリックします。

## 従来のUIから新しいUIへの切り替え

* ページの上部にある「**[!UICONTROL Switch to New UI]**」をクリックします。
