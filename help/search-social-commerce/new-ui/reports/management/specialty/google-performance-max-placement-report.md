# [!UICONTROL Google Performance Max Placement Report]

*パフォーマンス最大キャンペーンのみ*&#x200B;の[!DNL Google Ads] アカウントに適用できます

[!UICONTROL Google Performance Max Placement Report]には、指定されたアカウント内の[!DNL Google Ads] パフォーマンス最大キャンペーンのプレースメントに関するインプレッションデータが含まれています。 このレポートには、ターゲット URL別に含まれる各プレースメントのインプレッションが含まれます。 デフォルトでは、レポート期間中にプレースメントが削除された場合でも、指定されたデータ範囲で少なくとも1つのインプレッションを受け取ったプレースメントとターゲット URLの組み合わせごとに1行がデータに含まれます。 デフォルトでは、行はキャンペーンごとに昇順になっています。

このレポートを使用して、以下を行います。

* 広告に配信しているweb サイト、アプリ、[!DNL YouTube] チャネルを特定します。

* 無関係な配置やパフォーマンスの低い配置を特定して除外し、キャンペーン ROIを向上させます。

* 堅牢な除外リストを構築することで、広告がブランドの安全性が確保された環境でのみ表示されるようにできます。
