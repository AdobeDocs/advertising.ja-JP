# （新しいUI） [!UICONTROL Keywords] ビューについて

*Beta機能*

[!UICONTROL Manage] > [!UICONTROL Keywords] ビューには、次の2つのタブがあります。

* 「[!UICONTROL Keywords]」タブには、選択した広告主アカウントのフィルタービュー内のすべてのキーワードが一覧表示されます。

* 「[!UICONTROL Negatives]」タブには、選択した広告主アカウントのフィルター付きビュー内のすべての否定的なキーワードが一覧表示されます。
