# （新しい UI） [!UICONTROL Keywords] ビューについて

*Beta機能*

[!UICONTROL Manage]/[!UICONTROL Keywords] ビューには、次の 2 つのタブがあります。

* 「[!UICONTROL Keywords]」タブには、選択した広告主アカウントのフィルター済みビュー内のすべてのキーワードがリストされます。

* 「[!UICONTROL Negatives]」タブには、選択した広告主アカウントのフィルター済みビュー内のすべての負のキーワードがリストされます。
