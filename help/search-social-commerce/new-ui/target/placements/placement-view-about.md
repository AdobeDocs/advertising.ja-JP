# （新しい UI） [!UICONTROL Placements] ビューについて

*Beta機能*

[!UICONTROL Manage]/[!UICONTROL Placements] ビューには、次の 2 つのタブがあります。

* 「[!UICONTROL Placements]」タブには、選択した広告主アカウントのフィルター済みビューのすべてのプレースメントがリストされます。

* 「[!UICONTROL Negatives]」タブには、選択した広告主アカウントのフィルター済みビュー内のすべての負のプレースメントがリストされます。
