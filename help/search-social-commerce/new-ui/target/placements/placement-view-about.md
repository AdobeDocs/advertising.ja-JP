# （新しいUI） [!UICONTROL Placements] ビューについて

*Beta機能*

[!UICONTROL Manage] > [!UICONTROL Placements] ビューには、次の2つのタブがあります。

* 「[!UICONTROL Placements]」タブには、選択した広告主アカウントのフィルタービュー内のすべてのプレースメントが一覧表示されます。

* 「[!UICONTROL Negatives]」タブには、選択した広告主アカウントのフィルタービュー内のすべてのネガティブプレースメントが一覧表示されます。
