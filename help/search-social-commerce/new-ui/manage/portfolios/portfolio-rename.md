# （新しいUI）ポートフォリオ名の変更

*Beta機能*

完全なポートフォリオ設定を開かずに、ポートフォリオの名前をすばやく変更できます。

1. メインメニューで、**[!UICONTROL Manage]>[!UICONTROL Portfolios]**&#x200B;をクリックします。

1. ポートフォリオ行の上にカーソルを置き、**[!UICONTROL ...]>[!UICONTROL Rename]**&#x200B;をクリックします。

1. 名前を編集し、**[!UICONTROL Apply]**&#x200B;をクリックします。
