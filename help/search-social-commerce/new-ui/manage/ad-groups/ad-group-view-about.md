# （新しいUI） [!UICONTROL Ad Groups] ビューについて

*Beta機能*

[!UICONTROL Manage] > [!UICONTROL Ad Groups] ビューには、選択した広告主アカウントのフィルター処理されたビュー内のすべての広告グループが一覧表示されます。

* 行内から広告グループのステータスを削除または変更します。
