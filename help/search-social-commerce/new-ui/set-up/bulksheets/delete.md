# （新しいUI）アップロードされたバルクシートとエラーファイルの削除

バルクシートファイル、ランディングページ検証エラーファイル、およびその他のエラーファイルを手動で削除できます。 ファイルは、アップロードまたは生成されてから30日後に自動的に削除されます。

>[!TIP]
>
>バルクシートを投稿したがエラーを受け取った場合は、すべてのエラーが解決されるまでエラーファイルを削除しないでください。

1. メインメニューで、**[!UICONTROL Setup]** \> **[!UICONTROL Bulksheets]**&#x200B;をクリックします。

1. 削除する各ファイルの横にあるチェックボックスをオンにします。

1. 一括操作ツールバーで、**[!UICONTROL Delete]**&#x200B;をクリックします。

1. 確認メッセージで、**[!UICONTROL Confirm]**&#x200B;をクリックして削除を確認します。

   選択したバルクシートのいずれかが投稿するようにスケジュールされている場合は、確認する前に通知されます。 スケジュールされたバルクシートを削除すると、保留中の投稿がキャンセルされます。
