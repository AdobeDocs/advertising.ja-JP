# [!DNL Google Ads]変換値ルール設定

<!-- Go through all -->

| セクション | パラメーター | 説明 |
|---|---|---|
| [!UICONTROL Select Campaign] | [!UICONTROL Network] | 広告ネットワーク。 |
| | [!UICONTROL Account] | 広告ネットワークアカウント。 |
| | [!UICONTROL Campaign] | 広告キャンペーン。 |
| [!UICONTROL Conditions] > [!UICONTROL Primary Condition] | \[条件タイプ\] | （既存のルールの読み取り専用）値の調整をトリガーするために満たす必要がある条件のタイプ：<ul><li>**デバイス：**&#x200B;すべてまたは特定のデバイスタイプをターゲットにする</li><li>**場所：**&#x200B;すべての国と地域をターゲットにするか、特定の場所をターゲットにして除外します。</li><li>**オーディエンス：**&#x200B;すべてのオーディエンスまたは特定のオーディエンスをターゲットにする</li></ul>**注意：** アカウント内のすべてのルールは、同じタイプのプライマリ条件と（オプションの）セカンダリ条件を使用する必要があります。 例えば、ルール 1にプライマリ条件「Audience」とセカンダリ条件「Location」が含まれている場合、アカウント内の他のすべてのルールにはプライマリ条件「Audience」が必要です。 他のルールにセカンダリ条件が含まれる場合は、「場所」にする必要があります。 |
| | 状況タイプ/デバイス | （デバイスの状態のみ）対象となるデバイスタイプ：<ul><li>**すべてのデバイス** – すべてのデバイスタイプをターゲットにします。</li><li>**デバイスを選択** – ターゲットとする1つ以上のデバイスタイプを指定するには、**デスクトップ**、**モバイル**、および&#x200B;**タブレット**。</li></ul>条件内では、ブール演算子ORを使用して複数のターゲットが結合されるため、任意のオプションを満たして値の調整を開始できます。 例：\[ デバイスがモバイルまたはタブレット\]の場合は、1.5を追加します。 |
| | 状況タイプ/場所 | （ロケーション条件のみ）ロケーションのターゲットと除外条件：<ul><li>**すべての国と地域** – すべての国と地域をターゲットにするか、特定の地域をターゲットにして除外します。</li><li>**場所を入力** – 特定の場所をターゲットにして除外します。</li></ul><ul><li>場所またはサブロケーションを展開するには、名前の後にある>をクリックします。</li><li>ターゲットを追加するには、ターゲットを1回選択すると、緑色のチェックマークが表示されます。</li><li>ターゲットを除外するには、ターゲットを2回目に選択して、赤い&#x200B;**[!UICONTROL X]**&#x200B;を表示します。</li><li>ターゲットまたは除外を削除するには、次のいずれかの操作を行います。<ul><li>![列のアイテムの横にある](/help/search-social-commerce/assets/delete.png "削除")削除[!UICONTROL Selections]をクリックします。</li><li>チェックマークまたは[!UICONTROL X]が表示されないまで、ターゲットを選択します。</li></ul></li></ul>条件内では、ブール演算子ORを使用して複数のターゲットまたは除外条件を結合します。これにより、任意のオプションを満たして値の調整を開始できます。 例：\[場所がアルジェリアまたはチュニジア\]の場合は、1.5を追加します。 |
| | 条件タイプ/オーディエンス | （オーディエンス条件のみ）オーディエンスのターゲット：<ul><li>**すべてのオーディエンスセグメント** – すべての[!DNL Google Ads] オーディエンスセグメントをターゲットにします。</li><li>**オーディエンスセグメントを入力** – 特定の[!DNL Google Ads] オーディエンスセグメントをターゲットにします。</li></ul><ul><li>カテゴリまたはサブカテゴリをセグメントに展開するには、名前の後にある/をクリックします。</li><li>ターゲットを追加するには、ターゲットを選択します。</li><li>ターゲットを削除するには、ターゲットの選択を解除するか、選択列の項目の横にある![削除](/help/search-social-commerce/assets/delete.png "削除")をクリックします。</li></ul>条件内では、ブール演算子ORを使用して複数のターゲットまたは除外条件を結合します。これにより、任意のオプションを満たして値の調整を開始できます。 例：\[Audience is Bargain Travelers OR Family Vacationers\]の場合、1.5を追加します。<br><br>**注意：** オーディエンスターゲットを保存したら、追加のオーディエンスを追加できますが、[!DNL Google Ads] エディターの外で削除することはできません。 オーディエンスターゲットを削除する必要がある場合は、[!DNL Google Ads]に直接ログインし、[!DNL Google Ads] エディターを使用します。 |
| 条件/セカンダリ条件 | | （オプション、既存のルールの読み取り専用）値の補正をトリガーするために満たす必要がある条件のタイプ。 セカンダリ条件を含めると、セカンダリ条件はブール演算子ALLを使用してプライマリ条件に結合されるので、両方の条件を満たして値の調整を開始する必要があります。 例：\[Location is Algeria OR Tunisia\] AND \[Device is Mobile OR Tablet\]の場合、Add 1.5.<br><br>プライマリ条件の項目を参照してください。<br><br>**注意：** アカウント内のすべてのルールは、同じタイプのプライマリ条件と（オプションの）セカンダリ条件を使用する必要があります。 例えば、ルール 1にプライマリ条件「Audience」とセカンダリ条件「Location」が含まれている場合、アカウント内の他のすべてのルールにはプライマリ条件「Audience」が必要です。 他のルールにセカンダリ条件が含まれる場合は、「場所」にする必要があります。 |
| 値調整 | 値 | 調整タイプを指定し、正の値を入力します。<ul><li>**Add** – 指定した値を、渡されたコンバージョン値に追加します。 値は0より大きくなければなりません。</li><li>**Multiply** – 指定された値で渡されるコンバージョン値を乗算します。 値は0.5から10までである必要があります。</li></ul> |
