# [!UICONTROL Google Performance Max Placement Report]

*Performance MAX キャンペーンを使用する [!DNL Google Ads] アカウントにのみ適用*

[!UICONTROL Google Performance Max Placement Report] には、指定したアカウント内の Performance MAX キャンペーン [!DNL Google Ads] プレースメントのインプレッションデータが含まれます。 レポートには、ターゲット URL でインクルードされた各プレースメントのインプレッションが含まれます。 デフォルトでは、レポート期間中にプレースメントが削除された場合でも、指定したデータ範囲で少なくとも 1 つのインプレッションを受け取ったプレースメントとターゲット URL の組み合わせごとに 1 つの行がデータに含まれます。 デフォルトでは、行は campaign から昇順で並べられます。

このレポートを使用して、除外する URL を見つけ、Performance MAX キャンペーンのパフォーマンスを向上させます。
