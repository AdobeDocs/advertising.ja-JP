# 特殊レポートのレポート列

| 列 | 説明 |
|----|----|
| \[広告主固有のカスタム（派生）指標\] | 既存の指標から計算された、作成したカスタム指標の値。 |
| \[Advertiser-specific conversion metrics\] | 指定されたコンバージョン指標またはサイトエンゲージメント指標のコンバージョン数。 |
| [!UICONTROL Absolute Top Impression Rate Percent] | 検索結果の上部の最初の位置に表示された広告インプレッションの割合。 |
| [!UICONTROL Absolute Top Impression Share Lost To Budget Percent] | 日次または月次の予算が低すぎたため、検索結果の上位の最初の位置に広告が表示されなかった推定時間の割合。 |
| [!UICONTROL Absolute Top Impression Share Lost To Rank Percent] | 広告ランクが低いため、検索結果の上位の最初の位置に広告が表示されなかった時間の割合。 |
| [!UICONTROL Absolute Top Impression Share Percent] | 絶対上位の場所（オーガニック検索結果の上位の最初の広告）で受け取ったインプレッションを、上位の場所で受け取る資格のあるインプレッションの推定数で割った値。 |
| [!UICONTROL Account Name] | アカウント名。 |
| [!UICONTROL Account Number] | （[!UICONTROL MSA Ad Extension]件のレポート）広告ネットワークによって割り当てられたアカウント ID。 |
| [!UICONTROL Account Status] | （[!UICONTROL MSA Ad Extension] レポート）アカウントの状態：*[!UICONTROL Active]*、*[!UICONTROL Paused]*、または&#x200B;*[!UICONTROL Inactive]*。 |
| [!UICONTROL Ad Distribution] | 広告が配布された場所（[!UICONTROL Bing Ads Geo Report]、[!UICONTROL MSA Network Impression Share Report]、および[!UICONTROL MSA Network Performance Report]）: *[!UICONTROL Audience]* （オーディエンスネットワーク上） *、[!UICONTROL Search]* （検索結果ページ上）または&#x200B;*[!UICONTROL Content]* （所有するweb サイトまたは参加パートナーのweb サイト上）。[!DNL Microsoft] |

| [!UICONTROL Ad Group Name] | 広告グループ。 |

| [!UICONTROL Ad Title] | （[!UICONTROL MSA Ad Extension]件のレポート）広告タイトル。 |
| [!UICONTROL Advertiser] | 広告主名。 |
| [!UICONTROL Advertiser ID] | 広告主のSearch, Social, &amp; Commerce アカウントの数値ID。 |
| [!UICONTROL All Conversion Rate] | （[!UICONTROL MSA Ad Extension]件のレポート）コンバージョン率（パーセント）。 |
| [!UICONTROL All Conversions] | （[!UICONTROL Google AI Max Search Term Combination]および[!UICONTROL MSA Ad Extension]件のレポート） コンバージョンの合計数。 [!UICONTROL Google AI Max Search Term Combination Report]の場合、この値には、メインのコンバージョン列に含まれていないコンバージョンアクションが含まれます。 |
| [!UICONTROL All Conversions Qualified] | （[!UICONTROL MSA Ad Extension]件のレポート） コンバージョン数。 |
| [!UICONTROL All Conversions Value] | （[!UICONTROL Google AI Max Search Term Combination Report]）すべてのコンバージョンの値。 |
| [!UICONTROL All Conversions Value Per Cost] | （[!UICONTROL Google AI Max Search Term Combination Report]）すべてのコンバージョンの値を、広告インタラクション （クリック数や動画ビュー数など）の合計コストで割った値。 |
| [!UICONTROL All Cost Per Conversion] | （[!UICONTROL MSA Ad Extension]件のレポート） コンバージョンあたりのコスト。 |
| [!UICONTROL All Return on Ad Spend] | （[!UICONTROL MSA Ad Extension] レポート）総収益を支出で割った割合で表します。 |
| [!UICONTROL All Revenue Per Conversion] | （[!UICONTROL MSA Ad Extension]件のレポート） コンバージョンあたりの売上高。 |
| [!UICONTROL Asset] | （[!UICONTROL RSA Asset Report]） アセット。 |
| [!UICONTROL Asset ID] | （[!UICONTROL RSA Asset Report]）既存のアセットを識別する一意のID。 |
| [!UICONTROL Asset Performance Label] | （[!UICONTROL RSA Asset Report]） アセットの[[!DNL Google Ads]  パフォーマンス評価](https://support.google.com/google-ads/answer/9564897)。 |
| [!UICONTROL Asset Status] | （[!UICONTROL RSA Asset Report]） アセットの状態：*[!UICONTROL Enabled]*&#x200B;または&#x200B;*[!UICONTROL Enabled]* （削除）。 |
| [!UICONTROL Asset Type] | （[!UICONTROL RSA Asset Report]） アセットタイプ：*[!UICONTROL Creative Title]*&#x200B;または&#x200B;*[!UICONTROL Description]*。 |
| [!UICONTROL Assists] | （[!UICONTROL MSA Ad Extension]件のレポート）別のエンティティに関連付けられたコンバージョンにエンティティが貢献した回数。 |
| [!UICONTROL Audience ID] | オーディエンス IDの数値。 |
| [!UICONTROL Audience Name] | オーディエンス： 利用可能なオーディエンスには、[!DNL Google Ads]の顧客データベース、市場内、類似オーディエンスと、[!DNL [!DNL Microsoft Advertising]] カスタム、顧客マッチ、動的リマーケティング、市場内、リマーケティング、類似オーディエンスなどがあります。 |
| [!UICONTROL Audience Impression Lost To Budget Percent] | Microsoft Audience Networkで失ったインプレッションの割合。毎日または毎月の予算が低すぎたためです。 |
| [!UICONTROL Audience Impression Lost To Rank Percent] | 広告ランクが低いため、Microsoft Audience Networkで失ったインプレッションの割合。 |
| [!UICONTROL Audience Impression Share Percent] | [!DNL Microsoft Audience Network]のインプレッションの推定パーセンテージを、受信した可能性のあるインプレッションの合計数で割った値。 |
| [!UICONTROL Audience Status] | オーディエンスのステータス。 |
| [!UICONTROL Average CPC] | クリックあたりの平均コスト。 |
| [!UICONTROL Average CPM] | インプレッション数千件当たりの平均コスト （ミリ秒）。 |
| [!UICONTROL Average Position], [!UICONTROL Avg Position] | 指定された日付範囲内の広告の平均位置。<br><br>[!DNL Google Ads]および[!DNL LY Ads] キャンペーンの場合、このデータは2019年9月までのみ利用できます。 [!DNL Microsoft Advertising]の場合、このデータは2021年1月22日までのみ利用できます。 |
| [!UICONTROL Bid Match Type] | （[!UICONTROL MSA Ad Extension]件のレポート）入札されたキーワードに関連付けられた一致タイプ。 |
| [!UICONTROL Campaign Budget] | （[!UICONTROL Campaign Daily Impression Share Report]） キャンペーンの予算。 |
| [!UICONTROL Campaign Name] | キャンペーン。 |
| [!UICONTROL Campaign Start Date] | （[!UICONTROL Campaign Daily Impression Share Report]） キャンペーンの入札が行われた最初の日です。 |
| [!UICONTROL Campaign Status] | キャンペーンの状態：*[!UICONTROL Active]*、*[!UICONTROL Paused]*、*[!UICONTROL Ended]*、または&#x200B;*[!UICONTROL Deleted]*。 |
| [!UICONTROL Category] ([!UICONTROL 1st level] - [!UICONTROL 5th level]) | （[!UICONTROL AdWords Shopping Performance Report]）第1 レベルから第5 レベルの[!DNL Google]製品カテゴリ。 |
| [!UICONTROL Channel Type] | マーケティングチャネルの種類：*[!UICONTROL Search]*&#x200B;または&#x200B;*[!UICONTROL Content]*。 この列は、レポート設定の[!UICONTROL Search/Content]設定が「[!UICONTROL Combined]」の場合は含まれません。 |

| [!UICONTROL Clicks] | 指定された日付範囲内の広告のクリック数。 |
| [!UICONTROL Content IS% (Google)] | （[!DNL Google Ads]のみ；[!UICONTROL Campaign Daily Impression Share Report]） ディスプレイ/オーディエンスネットワーク上の広告に対して受け取ったインプレッションの数を、受け取る資格のあるインプレッションの推定数で割った数。 10%未満の割合は「`<10%`」、90%を超える割合は「`>90%`」と表示されます。 |
