# [!UICONTROL MSA Ad Extension by Keyword Report]

*[!DNL Microsoft Advertising]アカウントのみ*

このレポートは、各広告の拡張機能のインプレッション数、クリック数、支出、クリック単価、キーワード別のクリック数を示しています。 このレポートを使用して、各キーワードの広告拡張機能のパフォーマンスを比較します。

指標列のみを含み、ディメンションを含まないレポートを実行すると、生成されたレポートには、集計データを含む行と、すべての指標を含む行の2つの重複した行が含まれます。<!-- all metrics? -->

過去180日間のデータを表示できます。 Search, Social, &amp; Commerceは、[!DNL Microsoft Advertising]からデータをリアルタイムで取得します。データ量が多いレポートを生成するには、数分かかる場合があります。

>[!NOTE]
>
>レポートに大量のデータが含まれている場合、レポートは失敗する可能性があります。 このような場合は、より短い期間レポートを実行します。
