# [!UICONTROL Campaign Hourly Report]

[!UICONTROL Campaign Hourly Report]には、キーワードを含む1つ以上の[!DNL Google Ads] キャンペーンのコスト、クリック、および（オプションで）コンバージョンデータが含まれます。 デフォルトでは、データには、指定された日付範囲の時間ごとに該当するキャンペーンごとに1つの行が含まれ、行はキャンペーンごとに最初に昇順になり、次にポートフォリオごとに（該当する場合）昇順になります。

オプションで、広告主固有のラベル分類を含めることができます。

データは2021年5月15日から利用できます。<!-- [Later: You can view data for the previous NN days.] -->
