# [!UICONTROL Transaction Report]

[!UICONTROL Transaction Report]は、検索時および検索時の広告グループ内の現在のキーワードに対するトランザクションレベルの詳細を提供します
指定されたポートフォリオ（該当する場合）または広告アカウント内で、キャンペーンを表示したり、ソーシャルキャンペーンの広告セットを表示したりできます。 オプションで、コンバージョン指標を含めることができます。 このレポートの結果を広告主のトランザクションデータと比較して、Search, Social, &amp; Commerceが広告主のすべてのトランザクションをレポートしていることを確認できます。

デフォルトでは、レポートには、指定した日付範囲内で発生したトランザクションのデータが含まれます。 各トランザクションのデータには、各キーワードまたは広告/グループ/キャンペーン/ポートフォリオの組み合わせごとに、トランザクション ID、トランザクションとクリック/インプレッション時間、コンバージョンタイプ、コンバージョン指標、コンバージョン指標、コンバージョン指標の収益データ（1つの登録の場合は1つ、12 ドルの注文の場合は12など）が含まれます。 デフォルトでは、行はトランザクション時間で最初に昇順になり、次にクリック時間で昇順になります。

過去45日間のデータを表示できます。

>[!NOTE]
>
>* 異なる照合タイプを持つ複数の入札単位が同じトランザクション IDを持つ場合、トラッキング IDの収益は、指定したクリック日のクリック数に応じて分割されます。
>* レポートに、コンテンツ対応の検索キャンペーンまたはディスプレイまたはソーシャルキャンペーン（キーワードを含まない）の広告グループのデータが含まれる場合、完成したレポートの[!UICONTROL Keyword]列には、該当する広告グループ名（「広告グループのコンテンツ）あなたの広告グループ名」など）が含まれます。
